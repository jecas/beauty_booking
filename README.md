# Beauty Booking App

Ovo je kompletan sistem za zakazivanje tretmana za kozmetičke usluge, sa **frontend React** i **backend Django REST API**.

## Funkcionalnosti

- Rezervacija tretmana (Mobile i Desktop verzija)
- Push notifikacije nakon uspešne rezervacije
- AdminDashboard za blokiranje termina za privatne obaveze
- Termini se čuvaju u Django bazi (SQLite)
- Frontend direktno komunicira sa backend API-em
- React frontend je responsive i mobile-friendly
